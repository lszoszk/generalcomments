# Jurisprudence coverage — missing decisions

This folder lists the cases that appear in the [OHCHR JURIS
authority catalogue](https://juris.ohchr.org) but are **not yet
ingested** into the UN Human Rights Database corpus.

## Files

- **`missing-cases.csv`** — one row per missing decision, sortable by
  treaty / year. Columns: `symbol, treaty, country, decisionDate,
  typeOfDecision, jurisUrl, title`. Open the `jurisUrl` to see the
  OHCHR catalogue entry; the entry's download surface is what we
  attempted to fetch.

## Why these 65 cases are missing

We attempted to recover them from every UN-side source we could find.
The result, per source:

- **OHCHR docstore** (the primary surface) — for **43 of the 65**,
  the catalogue entry has an empty `downloads` dict in every
  language. The metadata record exists; no file is hosted.
- **3 cases** return HTTP 500 from OHCHR docstore (server-side
  regression on their end).
- **21 cases** have only non-English PDFs (French, Spanish,
  Russian). Ingesting non-English source as if it were English
  would damage the corpus; flagged for a future translation
  pipeline.
- `documents.un.org`, the new UN docs system, returns 404 for every
  old jurisprudence symbol we tested (it doesn't cover pre-2015
  records).
- `daccess-ods.un.org`, the legacy ODS, redirects to
  `documents.un.org/error` — UN deprecated the old system and
  didn't migrate jurisprudence.
- Refworld, CCPR Centre and the (archived) UN Minnesota Human
  Rights Library don't index these old records either.

The 65-case gap is **the floor of what's recoverable from public UN
online sources**. Closing it further requires either OHCHR
re-uploading the missing PDFs, OCR of printed UN compilations
(*Selected Decisions Under the Optional Protocol*), or a
translation step for the non-English-only entries.

## How to use this list

If you're a researcher who needs one of these decisions:

1. Click the `jurisUrl` column to open the OHCHR JURIS catalogue
   entry — you'll get the metadata (date, country, articles, type).
2. The actual decision text, when not online, is typically
   available in the relevant **annual report of the Committee** to
   the General Assembly (e.g., `A/<session>/40` for CCPR).
3. The UN library at the Palais des Nations in Geneva holds
   physical archives of all decisions.

If you're a maintainer who has found a recovery path for one of
these cases, please open a pull request adding the source — see
the audit script at `_docs_internal/jur_coverage_audit/` (private)
for the ingest plumbing.
